## 核心原则

### 1. 倾听 > 说话
- 70%倾听，30%说话
- 复述确认理解
- 提问深入了解

### 2. 共情回应
- 认可对方情绪
- 避免立刻给建议
- "我理解你的感受" > "你应该..."

### 3. 真诚 > 技巧
- 做真实的自己
- 过度使用技巧会被识破
- 真诚是最有力的武器

### 4. 适度留白
- 不需要填满每个沉默
- 对方也需要思考时间
- 适当的沉默是成熟的标志

## 聊天技巧模板

### 破冰开场 (Opener)

```
场景: 第一次见面/聊天

✅ 好例子:
- "我注意到你书架上有XX书，你最喜欢哪个作者？"
- "这个咖啡厅我来过很多次，你通常点什么？"
- "你今天看起来精神很好，有什么开心的事吗？"

❌ 避免:
- "在吗？" "你好"
- "你多大了？" "你赚多少钱？"
- 查户口式提问
```

### 深入话题 (Deepening)

```
场景: 想要更了解对方

✅ 好例子:
- "你最难忘的一段经历是什么？"
- "如果可以重来，你会选择什么职业？"
- "什么让你晚上睡不着觉？"

❌ 避免:
- 太快问隐私问题
- 连续追问
- 评价对方的选择
```

### 表达不同意见 (Disagreement)

```
场景: 观点不一致

✅ 好例子:
- "我理解你的观点，我的角度可能有些不同..."
- "这个很有趣，我从另一个角度来看..."
- "我以前也这么想过，但后来发现..."

❌ 避免:
- "你错了"
- "不是这样的"
- 争辩到底
```

### 表达拒绝 (Saying No)

```
场景: 需要拒绝对方

✅ 好例子:
- "谢谢你的邀请，但那段时间我已经有安排了"
- "你是个很好的人，但我目前想专注于工作"
- "我很喜欢和你聊天，但我想把关系定位在朋友"

❌ 避免:
- 直接消失/不回复
- 给出模糊的承诺
- 过度解释
```

### 得体接受拒绝 (Being Rejected)

```
场景: 被拒绝后

✅ 好例子:
- "理解，谢谢你的坦诚"
- "没关系，希望还能继续做朋友"
- "感谢你的直接，这需要勇气"

❌ 避免:
- 反复追问"为什么"
- 纠缠不清
- 表达愤怒或不满
```

## 常见错误警示

| 错误 | 问题 | 正确做法 |
|-----|------|---------|
| 查户口式提问 | 太直接像审问 | 开放式问题+分享自己 |
| 只顾自己说 | 不尊重对方 | 70%倾听 |
| 过度使用技巧 | 不真诚 | 真诚为本，技巧为辅 |
| 急于推进关系 | 给对方压力 | 尊重对方节奏 |
| 回避沉默 | 不成熟的表现 | 接受沉默是礼貌 |

## 进阶技巧

### 1. 镜像法 (Mirroring)
- 模仿对方的说话节奏和肢体语言
- 复述对方的话来确认理解
- "所以你的意思是...对吗？"

### 2. 开放式问题 (Open Questions)
- 避免是非题 (是/否)
- 用"什么"、"为什么"、"如何"
- 引导对方深入分享

### 3. 情绪标注 (Labeling)
- "你看起来有点..."
- "我能感受到你..."
- "这件事让你很..."

### 4. 适度沉默 (Strategic Silence)
- 等待对方填充空白
- 对方说话后停顿2-3秒
- 沉默有时是最好的回应
