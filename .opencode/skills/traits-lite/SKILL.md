## When to Use

使用此技能当你需要：
- 定义对话角色的性格特点
- 模拟不同沟通风格的人
- 丰富角色的表达方式
- 预测对方可能的反应

**触发词**: "人格画像", "沟通风格", "角色性格", "MBTI", "性格分析", "创建角色"

## 沟通风格类型 (4大类型)

### 1. 主导型 (D - Dominant)

**特点**: 直接、果断、结果导向、竞争性强

**说话风格**:
- 简短有力，少废话
- "直接说重点"
- 喜欢控制话题
- 决策快，可能冲动

**肢体语言**:
- 眼神直视，坚定
- 姿态开放但有压迫感
- 语速快

**如何应对**:
- 简洁回应，不要绕弯
- 给出选项让TA选择
- 尊重TA的时间

**模拟对话**:
```
对方: "这个方案行不行？一句话说清楚"
回应: "可以。成本降低20%，周期缩短2周。"
```

### 2. 影响型 (I - Influential)

**特点**: 热情、外向、乐观、社交能力强

**说话风格**:
- 语速快，语调高
- 喜欢用"太棒了！""太好了！"
- 思维跳跃，话题多
- 重视感觉和认可

**肢体语言**:
- 表情丰富
- 手势多
- 身体前倾，靠近

**如何应对**:
- 给予积极反馈
- 适当赞美
- 允许话题发散

**模拟对话**:
```
对方: "这个想法太棒了！我们一定要做！
你知道吗，我之前..."
回应: "确实很有潜力！我注意到[拉回重点]"
```

### 3. 稳健型 (S - Steady)

**特点**: 耐心、倾听、稳定、重视关系

**说话风格**:
- 语速慢，温和
- 很少说"不"
- 重视共识，不喜欢冲突
- 注重细节和流程

**肢体语言**:
- 放松，舒展
- 眼神温和
- 点头多

**如何应对**:
- 不要催促
- 提供安全感
- 逐步引导

**模拟对话**:
```
对方: "这个...我觉得可能还需要再考虑一下
毕竟涉及到很多人..."
回应: "我理解你的顾虑。我们一步一步来，
先看看大家的反馈？"
```

### 4. 谨慎型 (C - Conscientious)

**特点**: 分析、细节、逻辑、追求完美

**说话风格**:
- 语速慢，逻辑清晰
- 用数据说话
- 很少感情用事
- 重视准确和细节

**肢体语言**:
- 表情平静
- 保持距离
- 记录要点

**如何应对**:
- 提供数据和证据
- 不要催促决策
- 尊重TA的分析

**模拟对话**:
```
对方: "根据去年的数据，增长率是15%，
但考虑到季节性因素，我建议..."
回应: "数据支持你的结论。还有其他变量需要考虑吗？"
```

## 对话风格混合

### 常见混合类型

| 组合 | 特点 | 称呼 |
|-----|------|------|
| D+I | 果断但热情 | "执行型" |
| D+C | 果断且严谨 | "专业型" |
| I+S | 热情且稳定 | "亲和型" |
| S+C | 稳定且严谨 | "分析型" |
| D+S | 果断但耐心 | "领导型" |

### 混合型对话示例

```
D+C 型 (专业型):
"数据很清楚，方案A比B好30%。
下周一之前给我最终决定。"

I+S 型 (亲和型):
"大家都觉得这个方向很好！
我们慢慢来，不用着急～"
```

## 实用工具

### 快速判断对方类型

```
观察指标:
1. 说话快慢?
   快 → D或I   慢 → S或C

2. 是否关注情感?
   是 → I或S   否 → D或C

3. 决策速度?
   快 → D      慢 → C

4. 是否关注细节?
   是 → C      否 → D/I/S

快速判断:
- D: 快 + 不关注情感 + 快决策
- I: 快 + 关注情感
- S: 慢 + 关注情感
- C: 慢 + 不关注情感 + 关注细节
```

### 对应策略表

| 对方类型 | 沟通要点 | 避免 |
|---------|---------|------|
| D | 简洁、直接、结果 | 废话、绕弯 |
| I | 热情、积极、认可 | 太严肃、冷淡 |
| S | 耐心、安全、一致 | 催促、压力 |
| C | 数据、逻辑、细节 | 感觉、冲动 |

