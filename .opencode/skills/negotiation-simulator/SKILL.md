## Negotiation Framework

### 核心原则

#### 1. 信息不对称原则
- 谁先出价，谁就失去锚定优势
- 收集信息 > 展示信息
- 永远不要接受第一次报价

#### 2. BATNA (Best Alternative to Negotiated Agreement)
- 清晰知道你的最佳替代方案
- 没有BATNA = 没有谈判力
- 在开始前确定你的底线

#### 3. 价值创造 vs 价值争夺
- 双赢谈判 > 零和博弈
- 寻找共同利益
- 扩展谈判空间

### 谈判策略库

| 策略 | 适用场景 | 示例话术 |
|-----|---------|---------|
| **沉锚效应** | 对方先出价 | "根据市场行情..." |
| **条件交换** | 对方坚持 | "如果你能...我们可以..." |
| **时间压力** | 拖延策略 | "这个优惠只到本周五" |
| **拆解议题** | 大谈判僵局 | "我们先就X达成一致" |
| **沉默策略** | 对方出价后 | [等待，不急于回应] |
| **量价挂钩** | 供应商谈判 | "如果订单量翻倍呢？" |
| **价值锚定** | 薪资谈判 | "这个职位的市场价位是..." |

```
  我期望的薪资范围是 30-35K"

=== 反馈分析 ===
[AI会评估你的回应，并给出对方可能的反应和最佳策略]
```

## 常见错误警告

| 错误行为 | 后果 | 正确做法 |
|---------|------|---------|
| 接受第一次报价 | 损失15-30%价值 | 永远要求更多 |
| 先出价 | 失去锚定优势 | 让对方先出价 |
| 情绪化反应 | 失去谈判立场 | 保持专业冷静 |
| 只谈价格 | 忽略其他价值 | 扩展谈判议题 |
| 害怕说"不" | 被压价 | BATNA是你的武器 |

## 谈判话术模板库

### 开场阶段
```
- "感谢您抽出时间，我很想了解一下贵公司的主要考量"
- "在开始之前，我想先听听您对这个项目的期望"
```

### 僵局突破
```
- "我理解您的立场，让我们看看有没有其他方案"
- "这个问题我们先搁置，先讨论其他议题如何"
```

### 促成交易
```
- "我们都在正确的方向上，让我总结一下今天的共识"
- "基于今天的讨论，我们的最终方案是..."
```

### 应对压价
```
- "理解您的预算考量，我们能否讨论一下服务范围的调整"
- "这个价格是基于XX价值的，如果要调整，
  我们需要重新评估交付内容"
```

## 评估标准

每次练习后评估:

1. **信息收集** - 是否收集了足够信息？
2. **锚定控制** - 是否避免了不利锚定？
3. **价值创造** - 是否探索了双赢方案？
4. **BATNA运用** - 是否善用了替代方案？
5. **沟通技巧** - 是否保持了专业和积极？
