## When to Use

使用此技能当你需要：
- 练习不同的谈判风格

- 切换自己的谈判风格应对不同场景
- 学习风格组合的谈判策略

**触发词**: "谈判风格", "风格切换", "混合风格", "谈判模式", "谈判类型"

## 5大谈判风格

### 1. 竞争型 (Competitive)

**特点**: 强硬、赢者通吃、果断

**适用场景**:
- 一次性交易
- 对方有替代方案
- 需要快速决策

**表现方式**:
- 态度强硬，立场坚定
- 施压策略
- 快速推进

**话术示例**:
```
"这是我们的最终报价，没有谈判空间"
"你可以选择接受或放弃"
"如果不能接受这个价格，那就没有继续的必要"
```

**如何应对**:
- 不要让步太快
- 展示你的BATNA
- 保持冷静

### 2. 合作型 (Collaborative)

**特点**: 双赢导向、信任建设、长期关系

**适用场景**:
- 长期合作关系
- 需要双方满意
- 复杂问题解决

**表现方式**:
- 探索双方需求
- 创造性解决方案
- 强调共同利益

**话术示例**:
"我们来看看怎样做对双方都最有利"
"你的顾虑我理解，让我们一起想办法"
"我相信我们可以找到一个双方都满意的方案"

**如何应对**:
- 展示诚意
- 愿意让步
- 注重关系维护

### 3. 妥协型 (Compromising)

**特点**: 中间路线、公平分担、务实

**适用场景**:
- 双方实力相当
- 时间有限
- 需要快速达成协议

**表现方式**:
- 各退一步
- 寻找平衡点
- 务实而非理想

**话术示例**:
"我们各自让一步，各承担50%的成本"
"这样吧，你降10%，我们也接受"
"大家都退一步，找到一个中间点"

**如何应对**:
- 接受合理的让步
- 确保不是单方面损失
- 快速达成协议

### 4. 回避型 (Avoiding)

**特点**: 拖延、转移话题、保持距离

**适用场景**:
- 问题不重要
- 需要更多时间
- 情绪太激烈

**表现方式**:
- "我们改天再谈"
- 转移话题
- 不做承诺

**话术示例**:
"这个问题我们先放一放吧"
"让我再考虑一下"
"现在不是讨论这个的最佳时机"

**如何应对**:
- 判断是否需要现在解决
- 施加强度或等待
- 评估对方意图

### 5. 迁就型 (Accommodating)

**特点**: 退让、满足对方、维护关系

**适用场景**:
- 关系比利益重要
- 对方更需要
- 长期合作考量

**表现方式**:
- 主动让步
- 满足对方需求
- 牺牲短期利益

**话术示例**:
"好吧，按照你的方式来"
"这次我们可以接受"
"我们不希望因为这个影响关系"

**如何应对**:
- 珍惜对方的让步
- 长期关系投资
- 适当回报

## 风格选择决策树

```
开始谈判前，问自己：

1. 这个问题重要吗？
   否 → 回避型
   是 → 继续问

2. 对方重要吗？
   否 (一次性) → 竞争型
   是 → 继续问

3. 有时间深入谈判吗？
   否 → 妥协型
   是 → 继续问

4. 想要长期关系吗？
   否 → 竞争型/妥协型
   是 → 合作型
```

## 风格组合策略

### 常见组合

| 组合 | 场景 | 说明 |
|-----|------|------|
| 竞争+合作 | "硬中带软" | 立场坚定，但保持友好 |
| 合作+妥协 | "软中带实" | 追求双赢，但愿意让步 |
| 竞争+回避 | "以退为进" | 表面让步，实际拖延 |
| 合作+迁就 | "成人之美" | 注重关系，愿意付出 |

### 风格切换示例

```
场景: 价格谈判陷入僵局

初始风格 (竞争):
"这个价格已经是最优惠的了，不能再降"

切换到 (合作):
"不过，我们可以看看有没有其他方式，
比如延长付款周期来平衡？"

如果对方拒绝:
切换到 (妥协):
"这样吧，我们各退一步，价格降5%，
但付款周期不变"

如果对方依然强硬:
切换到 (回避):
"我们先休息一下，各自再考虑一下"
```

## 对手风格识别

### 快速识别表

| 行为特征 | 可能的风格 |
|---------|----------|
| 快速出价，态度强硬 | 竞争型 |
| 询问需求，关注感受 | 合作型 |
| "各退一步" | 妥协型 |
| "改天再谈" | 回避型 |
| 主动让步 | 迁就型 |

### 对手风格应对策略

| 对方风格 | 你的最佳风格 | 避免 |
|---------|-------------|------|
| 竞争型 | 竞争型/合作型 | 迁就型 |
| 合作型 | 合作型/迁就型 | 竞争型 |
| 妥协型 | 合作型/妥协型 | 回避型 |
| 回避型 | 竞争型/妥协型 | 迁就型 |
| 迁就型 | 合作型/竞争型 | 回避型 |

## 实战场景

### 场景1: 薪资谈判

```
对方HR: "预算有限，只能给到25K"

【选择你的风格】

A. 竞争型:
"这个薪资低于市场水平30%，
我们很难接受。除非有其他福利补充。"

B. 合作型:
"我理解预算限制。
我们看看能否在薪资结构上找到平衡？"

C. 妥协型:
"那这样吧，27K+奖金，
我们各退一步。"

D. 回避型:
"我需要考虑一下，
方便的话我明天给你答复。"
```

### 场景2: 供应商谈判

```
供应商: "这是最低价，无法再降"

【选择你的风格】

A. 竞争型:
"竞争对手的报价是XX，
你再考虑考虑"

B. 合作型:
"我们希望建立长期合作，
看看怎样对双方都合适"

C. 妥协型:
"如果订单量翻倍呢？
各承担一半的折扣"

D. 回避型:
"我需要和团队讨论一下，
下周给你最终答复"
```

### 场景3: 合作分成谈判

```
合作方: "我们要70%，因为我们负责推广"

【选择你的风格】

A. 竞争型:
"我们提供核心产品，
70%太高了，最多55开"

B. 合作型:
"让我们分析一下双方的投入和贡献，
找到一个公平的比例"

D. 迁就型:
"好吧，按你们的方式来吧"
```
