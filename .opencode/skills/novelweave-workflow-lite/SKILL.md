# novelweave-workflow-lite

// 轻量级对话场景创作工作流 - 创建角色、场景、对话
// 用于模拟复杂角色对话场景
// 激活词: "场景创作", "角色对话", "对话脚本", "创建场景"

## When to Use

使用此技能当你需要：
- 创建复杂的对话场景
- 设计不同角色的对话脚本
- 模拟多角色互动场景
- 规划特定情境的对话流程

**触发词**: "场景创作", "角色对话", "对话脚本", "创建场景", "写一个对话"

## 场景创建模板

### 模板1: 基础双角色对话

```
=== 场景模板 ===

【场景名称】: [名称]
【场景类型】: 商务 / 社交 / 约会 / 谈判 / 冲突
【背景设定】: [2-3句话描述场景]

【角色A】
- 姓名/身份: 
- 性格特点: 
- 沟通风格: 
- 目标/诉求: 

【角色B】
- 姓名/身份: 
- 性格特点: 
- 沟通风格: 
- 目标/诉求: 

【对话流程】
1. 开场 (角色A发起)
2. 发展 (角色B回应)
3. 高潮 (核心冲突/话题)
4. 结局 (达成/未达成)

```

```
  - 角色A: [身份+目标]

【场景设定】:

```

## 角色创作

### 沟通风格类型

| 类型 | 特点 | 典型表现 |
|-----|------|---------|
| **主导型 (D)** | 直接、果断 | "直接说重点" |
| **影响型 (I)** | 热情、外向 | "太棒了！我们..." |
| **稳健型 (S)** | 耐心、倾听 | "我理解..." |
| **谨慎型 (C)** | 分析、细节 | "数据支持..." |

### 角色模板

```
【角色模板】
姓名: [姓名]
年龄: [年龄]
身份: [职业/角色]
性格: [MBTI或简单描述]
沟通风格: [主导/影响/稳健/谨慎]
当前情绪: [中性/开心/焦虑/愤怒]
核心诉求: [TA想要什么]
隐藏信息: [可能不愿说的信息]
```

## 场景示例

### 示例1: 商务谈判场景

```
=== 场景: 供应商价格谈判 ===

【场景类型】: 商务谈判
【背景】: 你是采购经理，供应商价格高于预算15%

【角色A - 你】
- 身份: 采购经理
- 目标: 降价10%
- 策略: 量价挂钩、长期合作

【角色B - 供应商】
- 身份: 销售总监
- 目标: 维持价格
- 策略: 强调质量、服务

【对话流程】
1. 开场: 你指出价格超出预算
2. 发展: 供应商解释成本
3. 高潮: 双方僵持
4. 结局: [待定]
```

### 示例2: 约会场景

```
=== 场景: 初次约会 ===

【场景类型】: 约会
【背景】: 咖啡厅第三次见面

【角色A - 男方】
- 身份: 工程师
- 性格: 主导型，稍内向
- 目标: 增进了解

【角色B - 女方】
- 身份: 设计师
- 性格: 影响型，外向
- 目标: 观察对方是否合适

【对话流程】
1. 开场: 轻松话题（工作、爱好）
2. 发展: 深入话题（价值观、未来）
3. 高潮: 尝试肢体靠近
4. 结局: 道别 + 后续安排
```

## 使用方式

### 方式1: 创建场景

```
你: "创建一个谈判场景，甲方乙方，价格分歧"

AI → 生成完整场景模板
```

### 方式2: 继续对话

```
你: "角色A说：'这个价格我们无法接受'，请生成角色B的回应"

AI → 根据角色B的性格生成回应
```

### 方式3: 评估对话

```
你: "请用 conversation-practice 的标准评估这段对话：
[对话内容]"

AI → 从倾听、共情、真诚等维度评估
```

## 与其他技能组合

### 组合1: novelweave + conversation-practice

```
1. 用 novelweave 创建场景和角色
2. 用 conversation-practice 练习具体对话
3. 评估反馈
```

### 组合2: novelweave + traits

```
1. 用 novelweave 定义角色
2. 用 traits 的沟通风格类型丰富角色
3. 生成更真实的角色画像
```

### 组合3: 完整工作流

```
novelweave-workflow-lite → 创建场景
+ traits → 丰富角色人格
+ conversation-practice → 练习对话
+ negotiation-simulator → 如果是谈判场景
```
