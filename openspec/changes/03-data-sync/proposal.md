# 变更：数据同步功能

## 为什么
支持本地数据与云端数据同步，实现多设备访问。

## 什么会改变
- 首次登录时检测并导入本地数据
- 支持导出数据到本地
- 自动同步云端和本地数据
- 离线支持（本地暂存）

## 影响
- 修改文件：`worker/src/sync.js`
- 新增文件：`js/data-sync.js`（前端同步逻辑）

## 功能

| 功能 | 描述 |
|------|------|
| 首次登录导入 | localStorage → 云端 |
| 数据导出 | 云端 → JSON 下载 |
| 自动同步 | 增量同步，保持一致 |
| 离线支持 | 网络断开时本地暂存 |

## 同步策略

| 场景 | 策略 |
|------|------|
| 本地修改 | 自动上传 |
| 云端修改 | 自动下载 |
| 冲突 | 时间戳比较，以新为准 |
