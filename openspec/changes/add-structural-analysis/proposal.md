# 变更：新增结构化分析功能

## 为什么
当前仅支持情绪分析，识别负面内容并改写。部分用户希望在不改变原文的前提下，获得日记结构优化建议，让内容看起来更轻松、更有条理。

## 什么会改变
- 新增"结构优化"按钮，与现有的"保存并分析"并列
- 调用独立的 AI 分析，生成结构优化建议
- 分析结果以分段文字形式展示
- 用户可选择将优化建议保存为日记内容
- 后续分析基于保存后的内容进行

## 影响
- 受影响的规范：新增 `structural-analysis` 能力
- 受影响的代码：
  - `index.html` - 新增结构优化按钮
  - `js/api.js` - 新增结构化分析方法
  - `js/ui.js` - 新增结构优化弹窗和交互
  - `js/storage.js` - 结构化分析结果存储
  - `css/style.css` - 结构化分析展示样式
- 需新增存储字段：diary.structured_version（结构优化版本）
