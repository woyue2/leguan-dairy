# 变更：添加外部图片插入功能

## 为什么
用户希望在日记中插入外部图片链接，丰富日记内容表现形式。当前仅支持纯文本日记，缺少图片展示能力。

## 什么会改变
- 新增图片插入功能（弹窗输入 URL）
- 新增 Ctrl+V 粘贴图片链接支持
- 日记内容支持渲染外部图片
- 目录页显示图片缩略图
- 图片加载失败时显示占位符

## 影响
- 受影响的规范：新增 `external-image-insert` 能力
- 受影响的代码：
  - `js/ui.js` - 新增插入图片按钮、Ctrl+V 监听、图片占位符
  - `js/app.js` - 图片解析渲染逻辑
  - `css/style.css` - 图片样式、缩略图、占位符
- 新增文件：无
