# 🎯 File System API - 快速开始

## 一分钟上手

### 1. 启动应用

```bash
# 方式1: 使用启动脚本（推荐）
./start-dev.sh

# 方式2: 直接使用 Python
python3 -m http.server 8888
```

### 2. 访问应用

```
http://localhost:8888
```

### 3. 升级到文件存储

1. 看到 Header 显示：`本地存储 (点击升级到文件)`
2. **点击**这段文字
3. **确认**弹窗
4. **选择**保存位置（例如：桌面 → `我的日记.json`）
5. ✅ 完成！

### 4. 验证

1. **写**一篇日记
2. **点击**"保存"
3. **打开**你选择的文件（例如：桌面的 `我的日记.json`）
4. **查看**内容是否保存 ✅

## ❓ 常见问题

### Q: 第一次使用，我该怎么做？

1. 启动应用
2. 点击 Header 中的"本地存储 (点击升级到文件)"
3. 选择文件保存位置
4. 开始写日记！

### Q: 数据保存在哪里？

**A**: 你首次选择文件时决定的位置。
例如：
- Windows 桌面：`C:\Users\You\Desktop\diary.json`
- macOS 文档：`/Users/You/Documents/diary.json`

**提示**: Header 会显示文件名，但不显示完整路径（浏览器限制）。

### Q: 如何备份数据？

**最简单**：直接复制 `diary.json` 文件到其他位置！

### Q: 刷新页面会丢失数据吗？

**A**: 不会！应用会自动恢复文件连接。

### Q: 我想换一个存储位置？

1. 清除浏览器的 IndexedDB
   - 开发者工具 → Application → IndexedDB → 删除
2. 重新打开应用
3. 重新选择文件位置
